# CAPEDIG-COOP CA — Plateforme Web Institutionnelle

## Fichiers de règles
- Sécurité : @.claude/rules/security.md
- Stack et conventions : @.claude/rules/stack.md

## Projet
Plateforme web pour coopérative agricole ivoirienne.
Stack : React 19 + Tailwind CSS 3 + Vite (frontend) / PHP + MySQL (backend)

## Commandes
- Dev : npm run dev (depuis frontend/)
- Build : npm run build

## Règle absolue de design
- JAMAIS de bg-white ou shadow-md pour la navbar → toujours bg-capedig-brun
- JAMAIS de green-800, green-700 → toujours capedig-vert ou capedig-orange
- JAMAIS de bg-gray-900 pour footer → toujours bg-capedig-brun
- Palette officielle : capedig-brun (#3D2314) / capedig-orange (#D4641A) / capedig-vert (#2D6A4F) / capedig-beige (#F5F0EB)
